# minio-springboot-starter

#### 介绍
​	在springboot项目中集成minio对象存储。自动配置minioclient，并对基本的文件存储操作做了约定封装，使用更为简单明了。

#### 软件架构
软件架构说明

- jdk17

- spring-boot 2.7.4

- minio 8.4.5


#### 安装教程

1. 引入此maven依赖

   

#### 使用说明

1. 注入MinioFileStorage类，即可进行文件相关操作。亦可参见测试样例。
